Installation Steps

  1. Download the extension ZIP file
  2. Extract the ZIP to a folder on your computer
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode"
  5. Click "Load unpacked" and select the extension folder
  6. Start using AI Auto Marker with Ctrl+Shift+U!

Once installed, the extension icon will appear in your Chrome toolbar. You can pin it for easy access. The extension works immediately on any webpage with MCQs.

Note: For security reasons, Chrome may warn you about installing extensions in developer mode. This is normal and expected for locally installed extensions.
