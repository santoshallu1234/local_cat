# AI Auto Marker Chrome Extension

A powerful Chrome extension that captures web pages, extracts content, and uses AI to analyze and provide answers to questions. The extension is designed to help with copying AI-generated responses and extracted text directly to your clipboard.

## Key Features

- **One-Click Capture**: Press `Ctrl+Shift+U` to capture the current webpage
- **AI-Powered Analysis**: Automatically processes captured content using advanced AI
- **Smart Text Extraction**: Extracts relevant text and questions from web pages
- **Easy Copy Functionality**: Copy AI answers or extracted text to clipboard with a single click
- **Popup Display**: Shows results in a convenient popup window
- **Configurable Timing**: Adjust how long the popup stays visible
- **Developer Friendly**: Works in Chrome's developer mode for easy testing

## Installation Steps

1. **Download the Extension**:
   - Download the `ai-auto-marker-extension` folder as a ZIP file
   - Extract the ZIP file to a location on your computer

2. **Enable Developer Mode in Chrome**:
   - Open Chrome browser
   - Go to `chrome://extensions/`
   - Toggle "Developer mode" to ON (usually in the top-right corner)

3. **Load the Extension**:
   - Click "Load unpacked" button
   - Navigate to and select the `ai-auto-marker-extension` folder
   - The extension should now appear in your extensions list

4. **Verify Installation**:
   - Look for the extension icon in your Chrome toolbar
   - Pin it for easy access if needed

## How to Use the Extension

1. **Activate Capture**:
   - Navigate to any webpage you want to analyze
   - Press `Ctrl+Shift+U` to capture the page
   - Alternatively, click the extension icon in the toolbar

2. **View Results**:
   - A popup window will appear at the bottom-left of your screen
   - The popup displays AI-generated answers or extracted text
   - The window will automatically close after 3 seconds (default)

3. **Copy Content**:
   - Click the "Copy" button in the popup to copy the displayed text to your clipboard
   - The text is now ready to paste anywhere with `Ctrl+V`

## Changing Popup Display Time

1. **Access Settings**:
   - Click the extension icon in your Chrome toolbar
   - The settings page will open

2. **Adjust Timing**:
   - In the settings page, you'll see a dropdown for "Popup Close Timing"
   - Select your preferred time:
     - 1 second
     - 2 seconds
     - 3 seconds (default)
     - 5 seconds
     - 10 seconds
     - Never auto-close

3. **Save Settings**:
   - Click the "Save Settings" button
   - Your preference will be applied immediately
